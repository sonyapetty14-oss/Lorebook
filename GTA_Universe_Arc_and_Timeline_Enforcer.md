# GTA Universe Arc & Timeline Enforcer

## Purpose
Prevent continuity mistakes and keep campaigns consistent with GTA universe boundaries.

## Core Enforcement Rule
The **2D**, **3D**, and **HD** universes are separate continuities. Shared city names do not imply shared character history.

## Universe Buckets

### 2D Universe
- Games: GTA (1997), GTA: London 1969, GTA: London 1961, GTA 2.
- Use when tone is retro/arcade and continuity is minimalistic.

### 3D Universe
- Games: GTA III, Vice City, San Andreas, Liberty City Stories, Vice City Stories.
- Use for stylized crime-saga campaigns and classic-era references.

### HD Universe
- Games: GTA IV, The Lost and Damned, The Ballad of Gay Tony, Chinatown Wars, GTA V, GTA Online.
- Use for modern, satirical, and more grounded continuity-driven arcs.

## Timeline Discipline Checklist
Before locking a story beat, verify:
1. Correct universe bucket.
2. Correct city version for that universe.
3. Character existed in that continuity.
4. Technology/culture matches timeframe.
5. Faction power level matches the selected game period.

## Conflict Resolution Protocol
If a proposed event conflicts with canon:
- Option A: Reframe it as rumor/urban legend.
- Option B: Recast as alternate-universe reinterpretation.
- Option C: Move the event to a custom non-canon server branch.

## Output Format for Lore Decisions
Use this structure in moderation workflows:
- **Claim:** (what a player/writer proposes)
- **Status:** Canon / Plausible-Inferred / Non-Canon
- **Universe:** 2D / 3D / HD
- **Reason:** concise explanation
- **Allowed Rewrite:** corrected version if needed

## Example
- **Claim:** "Niko met CJ in person before GTA V events."
- **Status:** Non-Canon
- **Universe:** Cross-universe conflict (3D vs HD)
- **Reason:** CJ is 3D-universe continuity; Niko is HD-universe continuity.
- **Allowed Rewrite:** "An HD-era character references CJ as a mythologized figure in pop culture."
